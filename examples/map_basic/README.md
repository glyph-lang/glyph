# Map Example

This example demonstrates basic `Map<K, V>` usage with add/update/get/has/keys/vals/del.

## Files
- `glph.toml` — package definition
- `src/main.glyph` — example program

## Build and run
From this directory (using the `glyph` executable):

```bash
glyph run src/main.glyph
```

Or build the binary explicitly:

```bash
glyph build src/main.glyph --emit exe
./map_basic
```
